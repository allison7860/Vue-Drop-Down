

Lets download/upgrade:

Update Node.js
	sudo npm cache clean -f
	sudo npm install -g n
Update npm
	sudo npm install npm@latest -g

Create a central folder for all Vue projects (global)
Then install inside:
	sudo npm install -g vue-cli
Create a project:
	vue init webpack vue-webpack
	--> cd vue-webpack
	sudo npm install
	(npm run dev)

Server running on localhost:8080

Changing Permissions
	sudo chmod a+rw ....
